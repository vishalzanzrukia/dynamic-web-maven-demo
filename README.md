It basically contains the dynamic web application environment done with Eclipse, so that it can be quite handy to take this demo and just run it. :)

#### Dependencies Used,

* **Hibernate**
* **Spring-framework**
* **Spring-security**
* **Spring-social**
* **JSP-Servlet API**
* **Junit**

#### Environment Used

* **Eclipse Mars**
* **Maven 3.2**
